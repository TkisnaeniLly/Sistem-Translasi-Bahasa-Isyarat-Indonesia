Cara Menggunakan Sistem Translate

Sistem ini membantu kamu menerjemahkan bahasa isyarat menjadi teks secara otomatis. Kamu hanya perlu menggunakan kamera secara langsung.

Langkah Penggunaan:

  • Buka menu Translate
  • Hidupkan kamera
  • Tunggu proses translasi
  • Hasil translasi muncul otomatis
